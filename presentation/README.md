# DeepAsk presentation pack

Files:

- `deepask-5min-slides.odp` - LibreOffice Impress presentation
- `deepask-5min-slides.html` - browser-based slide deck
- `deepask-5min-manuscript.md` - 5 minute timed speaking script
- `screenshots/` - screenshots captured from the local DeepAsk app

Open the `.odp` file in LibreOffice Impress. The HTML version is kept as a browser backup.

The deck is timed for about 5 minutes:

- 0:00-0:30 - opening
- 0:30-1:10 - survey problem
- 1:10-1:55 - participant flow
- 1:55-2:45 - governance and workpack
- 2:45-3:40 - Mind Hive
- 3:40-4:35 - organiser surface
- 4:35-5:00 - close
